# wcprediction-mcp

A [Model Context Protocol](https://modelcontextprotocol.io) (MCP) server that exposes a
World Cup 2026 prediction game's data as tools an AI agent can call.

Built as a hands-on learning project — the commit history is the step-by-step build.

## Status

Work in progress. Starting with a single `list_teams` tool, then growing toward
fixtures, predictions, standings, and accuracy tools.
